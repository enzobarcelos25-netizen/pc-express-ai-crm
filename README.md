# PC Express AI CRM

Sistema web MVP para captar clientes de suporte em TI, fazer diagnostico automatizado, gerar recomendacoes e organizar leads em um painel simples.

## Stack

- React + Vite
- JavaScript
- LocalStorage
- Integracao WhatsApp

## Rodar localmente

```bash
npm install
npm run dev
```

## Telas

- Landing page para conversao
- Diagnostico com perguntas guiadas
- Resultado com recomendacao e link do WhatsApp
- Painel admin com leads e status

## Evolucao

- Backend Node.js + Express
- PostgreSQL + Prisma
- Autenticacao
- IA com OpenAI API
- Dashboard com metricas
